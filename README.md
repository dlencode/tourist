# seon-tourist

В первую очередь установить все зависимости:  
`npm install -g gulp-cli bower`  
`npm install`  
`bower install`  

---

Для запуска локального сервера и разработки:  
`gulp serve`


Собрать и минифицировать проект:  
`gulp`

Все соберется в папку dist/
